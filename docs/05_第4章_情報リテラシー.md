## 4-1. インターネットリテラシー

### インターネットリテラシーとは
インターネットを適切に利用するために必要な能力の総称。以下の5要素で構成:

| 要素 | 内容 |
|---|---|
| **テクノロジーの理解** | 技術の仕組みを基本的に理解する力 |
| **情報リテラシー** | 情報を正しく評価・活用する力 |
| **セキュリティとプライバシー** | 自分と他者の情報を守る力 |
| **デジタル市民権** | デジタル社会で責任ある行動をとる力 |

## 4-2. セキュリティとプライバシー

### フィッシング詐欺とその派生

| 攻撃手法 | 説明 |
|---|---|
| **フィッシング詐欺** | 偽のWebサイトやメールで個人情報を騙し取る |
| **スミッシング** | SMSを使ったフィッシング |
| **ヴィッシング** | 音声通話（Voice）を使ったフィッシング |
| **スピアフィッシング** | 特定の個人・組織を狙った標的型フィッシング |

### マルウェア関連

| 用語 | 説明 |
|---|---|
| **マルウェア** | 悪意のあるソフトウェアの総称 |
| **アンチウイルスソフトウェア** | マルウェアを検出・除去するソフト |
| **ランサムウェア** | データを暗号化し身代金を要求するマルウェア |

### ソーシャルエンジニアリング攻撃
人間の心理的な弱点を突いて情報を入手する手法:

| 手法 | 説明 |
|---|---|
| **ソーシャルエンジニアリング攻撃** | 技術ではなく人間の心理を悪用する攻撃の総称 |
| **ベイト攻撃** | USBメモリ等の「餌」を使って誘導 |
| **ブラックメール** | 脅迫により情報や金銭を要求 |
| **プレテキスト** | 偽の口実（身分詐称等）で情報を引き出す |

### その他のセキュリティ脅威
- **悪意のあるQRコード**: 偽のQRコードでフィッシングサイトに誘導
- **Wi-Fiに潜む罠**: 偽のWi-Fiスポットで通信を傍受
- **アップロードサービスに潜む詐欺**: ファイル共有サービスを悪用した攻撃
- **不適切なコンテンツへのWebアクセス**: 有害サイトへの意図しないアクセス

### プライバシー設定
- SNS等のプライバシー設定を適切に管理する重要性

### 生成AIの技術的発展に潜む脅威
- 生成AIを悪用したフィッシングメールの高度化
- ディープフェイクによるなりすまし
- AIを使った攻撃の自動化

## 4-3. 個人情報保護の観点

### 個人情報保護法の体系

| 用語 | 説明 |
|---|---|
| **個人情報保護法** | 個人情報の適正な取り扱いを定めた法律 |
| **改正個人情報保護法** | デジタル社会に対応した改正版 |
| **個人情報保護委員会** | 個人情報保護を監督する独立機関 |
| **個人情報取扱事業者** | 個人情報データベースを事業に利用する者 |

### 個人情報の分類（超重要・混同注意）

| 分類 | 説明 | 例 |
|---|---|---|
| **個人識別符号** | 特定個人を識別できる符号 | 指紋、顔認証データ、マイナンバー |
| **要配慮個人情報** | 不当な差別・偏見が生じうる情報。取得に本人同意が必要 | 人種、信条、病歴、犯罪歴 |
| **機微（センシティブ）情報** | 金融分野等で特に慎重な取り扱いが求められる情報 | 健康情報、性生活情報 |
| **匿名加工情報** | 特定個人を識別できないよう加工した情報 | 統計データ等 |

### 重要テクニック
- **マスキング**: 個人情報の一部を隠す処理（例: 名前を「***」に置換）

### 生成AI活用における個人情報の取り扱い
- 生成AIへの入力に個人情報を含めないよう注意
- 学習データに個人情報が含まれるリスクの認識

## 4-4. 制作物に関わる権利

### 知的財産権の体系

```
知的財産権
├── 著作権: 創作物を保護（登録不要で自動発生）
├── 特許権: 発明を保護（出願・登録が必要）
├── 商標権: ブランド名・ロゴを保護（出願・登録が必要）
└── 意匠権: デザインを保護（出願・登録が必要）
```

### 人格権関連

| 権利 | 説明 |
|---|---|
| **肖像権** | 自分の容姿を無断で撮影・公開されない権利 |
| **パブリシティ権** | 著名人の氏名・肖像が持つ経済的価値を保護する権利 |

### 不正競争防止法

| 用語 | 説明 |
|---|---|
| **不正競争防止法** | 事業者間の公正な競争を確保するための法律 |
| **営業秘密** | 秘密管理性・有用性・非公知性の3要件を満たす情報 |
| **限定提供データ** | 特定の者に提供される技術的管理されたデータ |

### AI生成物に関する権利（頻出）
- **著作権侵害**: AI生成物が既存の著作物に類似する場合のリスク
- **名誉毀損**: AI生成物が他者の名誉を傷つける可能性
- **AI生成物に関する権利**: AIが自律的に生成した物に著作権は原則発生しない
- **AI生成物の著作権の所在**: 人間の創作的関与がある場合は著作権が認められる可能性
- **AI生成物が既存の権利を侵害する可能性**: 学習データに含まれる著作物との類似性リスク

## 4-5. AIを取り巻く理念と原則・ガイドライン

### AI社会の基本理念（3つ）

| 理念 | 英語 | 内容 |
|---|---|---|
| 人間の尊厳が尊重される社会 | **Dignity** | 人間の尊厳を最優先 |
| 多様な背景を持つ人々が多様な幸せを追求できる社会 | **Diversity & Inclusion** | 多様性と包摂性 |
| 持続可能な社会 | **Sustainability** | 環境・社会の持続可能性 |

### AI社会原則（7つ）
1. **人間中心の考え方**
2. **安全性・公平性**
3. **プライバシー保護**
4. **セキュリティ確保**
5. **透明性**
6. **アカウンタビリティ**（説明責任）

### 共通の指針（高度なAIシステムに関係する事業者向け）
上記に加えて:
7. **教育・リテラシー**
8. **公正競争確保**
9. **イノベーション**

### AIガバナンスの構築
- **環境・リスク分析**: AIシステムのリスクを評価
- **AIガバナンス・ゴール**: 組織のAI活用における目標設定
- **AIマネジメントシステム**: AIの開発・運用を管理する体制

### AIの事業活動を担う3つの主体

| 主体 | 英語 | 役割 |
|---|---|---|
| **AI開発者** | AI Developer | AIモデル・システムを開発 |
| **AI提供者** | AI Provider | AIサービスを提供 |
| **AI利用者** | AI Business User | AIを業務で利用 |
