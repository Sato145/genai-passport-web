# 正解と解説

## 第1章

| 問 | 正解 | 解説 |
|---|---|---|
| 1 | B | 1956年のダートマス会議でジョン・マッカーシーが「AI」を提唱 |
| 2 | C | 機械学習はデータからパターンを自動学習する。ルールベースは人間がルールを定義 |
| 3 | B | 教師なし学習はラベルなしデータから構造を発見。クラスタリングや次元削減が代表例 |
| 4 | C | 過学習（オーバーフィッティング）は訓練データへの過度な適合 |
| 5 | B | ドロップアウトはランダムにノードを無効化。正則化はペナルティを課す別の手法 |
| 6 | B | 現在のAIはすべて特定タスク特化の弱いAI（ANI）。AGIは未実現 |
| 7 | B | 第二次AIブーム=エキスパートシステム。第一次=探索・推論、第三次=ディープラーニング |
| 8 | C | レイ・カーツワイルが2045年問題を提唱。ヴィンジはシンギュラリティ概念の提唱者 |
| 9 | B | AI効果は実現した技術を「AIではない」と過小評価する心理現象 |
| 10 | B | 転移学習は学習済みモデルの知識を別タスクに再利用。ファインチューニングは追加学習 |
| 11 | B | ノーフリーランチ定理は「万能アルゴリズムは存在しない」という定理 |
| 12 | B | ディープラーニング=多数の層を持つニューラルネットワークによる学習 |
