# 第2章 生成AI（ジェネレーティブAI）

---

## 2-1. 生成AIの誕生まで

### 生成AI（ジェネレーティブAI）とは
- 学習したデータのパターンから新しいコンテンツ（テキスト、画像、音声等）を生成するAI

### 生成モデルの系譜

```
ボルツマンマシン → 制約付きボルツマンマシン
    ↓
自己回帰モデル → CNN → VAE → GAN → RNN → LSTM → Transformer
```

### 各モデルの要点

| モデル | 正式名称 | 特徴 |
|---|---|---|
| **ボルツマンマシン** | - | 確率的にデータの分布を学習する初期の生成モデル |
| **制約付きボルツマンマシン** | Restricted Boltzmann Machine | ボルツマンマシンの接続を制限し学習を効率化 |
| **自己回帰モデル** | Autoregressive Model | 過去のデータから次のデータを順番に予測 |
| **CNN** | 畳み込みニューラルネットワーク | 画像認識に強い。**畳み込み**処理で特徴を抽出 |
| **VAE** | 変分自己符号化器 | **エンコーダ**でデータを**潜在ベクトル**に圧縮→**デコーダ**で復元。**ノイズ**を加えて多様な生成 |
| **GAN** | 敵対的生成ネットワーク | **生成器**と**識別器**が競い合って学習。リアルな画像生成に強い |
| **RNN** | 回帰型ニューラルネットワーク | 時系列・**シーケンスデータ**に対応。**隠れ層**・**リカレント層**で過去の情報を保持 |
| **LSTM** | 長・短期記憶 | RNNの長期依存性問題を解決。長い文脈を記憶可能 |

### Transformerモデル（最重要）
- 2017年にGoogleが発表した「Attention Is All You Need」論文で登場
- **Attention層**: 入力の各要素間の関連度を計算
- **自己注意力（Self-Attention）**: 文中の各単語が他の全単語との関係を同時に計算
- **Attention Mechanism**: 重要な情報に「注意」を向ける仕組み
- **位置エンコーディング**: 単語の順序情報を付与（RNNと違い並列処理が可能）
- **アーキテクチャ**: エンコーダ・デコーダ構造

### Transformer登場以後の派生モデル

| モデル | 開発元 | 特徴 |
|---|---|---|
| **GPTモデル** | OpenAI | デコーダのみ使用。テキスト生成に特化 |
| **BERTモデル** | Google | エンコーダのみ使用。文脈理解に特化 |
| **RoBERTa** | Meta | BERTの学習手法を改良 |
| **ALBERT** | Google | BERTを軽量化（a Lite BERT） |

### BERTの学習手法
- **MLM（Masked Language Model）**: 文中の一部を隠して予測させる
- **NSP（Next Sentence Prediction）**: 2つの文が連続するかを予測

## 2-2. ChatGPT

### ChatGPTとは
- OpenAIが開発した対話型テキスト生成AI
- GPTモデルをベースに、人間のフィードバックで調整

### GPTモデルの進化

| モデル | 特徴 |
|---|---|
| **GPT-1** | Transformerベースの最初の言語モデル。**自然言語処理（NLP）** |
| **GPT-2** | パラメータ数を大幅増加。高品質なテキスト生成 |
| **GPT-3** | 1750億**パラメータ**。Few-Shot学習が可能に |
| **InstructGPT** | **RLHF**で人間の指示に従うよう調整 |
| **GPT-3.5** | ChatGPTの初期モデル |
| **GPT-4** | **マルチモーダル**対応（テキスト＋画像入力） |
| **GPT-4o** | 高速・低コスト版のGPT-4。音声対話にも対応 |
| **GPT-o1** | 推論能力を強化したモデル（思考の連鎖） |
| **GPT-o3** | o1の後継。より高度な推論 |
| **GPT-o4** | oシリーズの最新推論モデル |
| **GPT-4.1** | GPT-4の改良版。コーディング能力向上 |
| **GPT-5** | 次世代フラッグシップモデル |

### 重要な技術概念
- **データセット**: モデルの学習に使用するデータの集合
- **RLHF（Reinforcement Learning from Human Feedback）**: 人間のフィードバックを用いた強化学習
- **アライメント（Alignment）**: AIの出力を人間の意図や価値観に合わせること
- **ファインチューニング**: 学習済みモデルを特定タスク向けに追加学習
- **ハルシネーション（Hallucination）**: AIがもっともらしいが事実と異なる情報を生成する現象

### ChatGPTの主要機能・サービス
- **Code Interpreter**: コードの実行・データ分析が可能な機能
- **GPTs**: ユーザーがカスタムChatGPTを作成できる機能
- **Sora**: OpenAIの動画生成AI
- **Operator**: AIが自律的にWebブラウザを操作するエージェント
- **Codex**: コード生成に特化したAIモデル/サービス
- **Image Generation**: ChatGPT内の画像生成機能
